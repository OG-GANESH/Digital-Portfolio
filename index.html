Elevator Ad Hoarding System

This project involved creating a system where advertisements displayed on hoardings near or inside an elevator could be changed based on the elevator's current floor or direction of travel.

Goal: To explore dynamic advertising synchronized with physical movement.

Technologies Used:

  • Arduino Microcontroller
  • C/C++ Programming
  • Floor Sensors (e.g., Limit Switches, IR Sensors)
  • LCD Display / LED Matrix
  • Basic Circuit Design

Outcome: Successfully demonstrated the concept of triggering different outputs based on sensor input mimicking elevator movement.

Laser Security System

This project implements a simple perimeter security system. A laser beam is aimed at a light sensor (LDR). If the beam is broken, the system detects the change and triggers an alarm.

Goal: To create a basic intrusion detection mechanism using common electronic components.

Technologies Used:

  • Arduino Microcontroller
  • C/C++ Programming
  • Laser Diode
  • Light Dependent Resistor (LDR)
  • Buzzer/LED (for alarm)

Outcome: Successfully built a functional prototype that triggers an alert when the laser beam path is interrupted.

Line Following Robot

This project involved building a small autonomous robot that uses sensors to detect and follow a black line on a contrasting surface.

Goal: To understand basic robotics principles, sensor integration, and motor control algorithms.

Technologies Used:

  • Arduino Microcontroller
  • C/C++ Programming
  • IR Sensor Array
  • DC Motors & Motor Driver (e.g., L293D)
  • Robot Chassis

Outcome: Developed a robot capable of autonomously navigating a predefined line path using sensor feedback and motor adjustments.
